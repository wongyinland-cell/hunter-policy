# Privacy Policy for Hunter_Agent

This application (Hunter_Agent) is a self-hosted tool designed for personal content management.

## Data Collection

We do not collect or store any personal data on external servers. All TikTok authentication tokens and media files are stored locally on the user's hardware (MacBook Air).

## Data Usage

We only use TikTok APIs to upload user-generated content to the user's own TikTok account as drafts or private videos.

## Third-party Sharing

We do not share, sell, or trade any user information with third parties.

## Data Security

Security is maintained via local environment variables (.env) and local storage.
